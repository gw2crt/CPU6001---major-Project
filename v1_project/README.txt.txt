Finaly Year Project - Gareth Weston
-----------------------------------

v1_gen101_conn.js 

File contains the code to connect to the genuino 101 via BLE comments are made in code. Execute with Node.js!!! 

v1_obd_conn.py

File contains the code to connect the OBD_II bluetooth adapter to the rasp pi. Device must be paired before running this file or connection will fail.
Execute with Python!!!

## OBD_II device identity  - 00:1D:A5:68:98:8B - rfcomm connect hci0 00:1D:A5:68:98:8B ##

v1_gen101_acc

Folder contains the code for the genuino 101 using the Arduino IDE. Code is commented for explanation.
